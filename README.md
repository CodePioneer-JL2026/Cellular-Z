# Cellular-Z
Know more about wireless! An open-source tool focused on telecom cell, signal quality and WIFI network information (source code version).

## üì± Core Features
1. **Dual SIM mobile network information viewing**: SIM card information, serving cell information, serving cell signal quality, and neighboring cell information can be obtained on some phones.
2. **WIFI information monitoring**: View connected WIFI info, nearby WIFI list, and WIFI channels (2.5 and 5GHz frequency bands).
3. **Current location information display**: Check real-time positioning-related data.
4. **Device information query**: One-stop viewing of battery info, hardware info and system info.
5. **Network speed test**: Built-in SpeedTest function to detect network speed.
6. **Test trajectory recording**: Support recording and viewing of indoor coverage and test trajectories.
